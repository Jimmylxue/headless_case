# 前端自动化 - case

我们通过写一些 node 脚本，可以实现一些简单的前端自动化功能。如果将这些自动化融入公司的一些重复的比较大的业务场景，岂不是提升效率的最佳实践场景？

本项目将会沉淀一些自动化脚本案例。
